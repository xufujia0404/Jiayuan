# Common Scene Creation Issues

## Capture Rounds
- ALWAYS limit capture-and-fix rounds to 3 (three).

## Strategy
- Modify existing scripts rather than regenerating from scratch.

## Capture Tools
- 3D: Use `CaptureMultiAngleSceneView`.
- 2D: Use `Capture2DScene`.

## Placement Verification
- Check for sinking, overlapping, or floating objects one by one.
- Ensure objects are inside their containers.
