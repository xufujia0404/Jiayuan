## 问题描述

### 问题表现
Player角色初始状态插入到地面以下，导致游戏开始时角色位置错误。

### 触发条件
- 使用CapsuleCollider作为Player的碰撞体
- 需要角色站在地面上

## 解决方案

### 关键步骤
1. 精确计算Player初始位置和Collider配置
2. 确保碰撞体底部刚好在地面上
3. 使用公式验证：碰撞体底部Y = Player位置Y + Center.Y - Height/2

### 关键代码/命令

**配置示例**：
- Player位置：(0, 0.75, 0)
- CapsuleCollider配置：
  - Radius: 0.3
  - Height: 1.5
  - Center: (0, 0, 0)

**验证计算**：
```
碰撞体底部Y = 0.75 + 0 - 1.5/2 = 0
刚好在地面上（Y=0）
```

### 最终方案
Player初始位置和Collider配置需要精确计算，确保碰撞体底部刚好在地面上。使用CapsuleCollider时，Center和Height需要配合调整。
