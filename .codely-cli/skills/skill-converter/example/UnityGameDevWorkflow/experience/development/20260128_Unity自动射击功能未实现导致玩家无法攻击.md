## 问题描述

### 问题表现
游戏运行时，玩家无法发射子弹：
- 按键无反应（自动射击应该不需要按键）
- 控制台没有子弹生成的日志
- PlayerController代码存在Shoot()方法但从未执行
- 子弹Prefab引用未配置

### 触发条件
- PlayerController定义了Shoot()方法但没有在Update()中调用
- 缺少自动射击的时间控制逻辑
- bulletPrefab字段为null或未在Inspector中赋值
- 没有使用协程或时间累加实现定时射击

## 解决方案

### 关键步骤
1. 在PlayerController中添加bulletPrefab字段引用
2. 添加fireRate字段控制射击频率
3. 添加_nextFireTime字段记录下一次射击时间
4. 在Update()中添加HandleShooting()方法实现自动射击
5. 在Inspector中配置bulletPrefab引用

### 关键代码/命令
```csharp
using UnityEngine;

public class PlayerController : MonoBehaviour
{
    [SerializeField] private GameObject _bulletPrefab;
    [SerializeField] private float _fireRate = 0.5f;

    private float _nextFireTime;

    private void Update()
    {
        HandleShooting();
    }

    private void HandleShooting()
    {
        // 自动射击：每隔fireRate秒发射一次子弹
        if (Time.time >= _nextFireTime)
        {
            Shoot();
            _nextFireTime = Time.time + _fireRate;
        }
    }

    private void Shoot()
    {
        if (_bulletPrefab == null)
        {
            Debug.LogError("Bullet prefab is not assigned!");
            return;
        }

        // 在玩家位置实例化子弹
        Vector3 spawnPosition = transform.position;
        GameObject bullet = Instantiate(_bulletPrefab, spawnPosition, Quaternion.identity);

        Debug.Log($"[PlayerController] Shot bullet at position: {spawnPosition}");
    }
}
```

### 最终方案
自动射击功能需要完整的实现流程：
1. **配置子弹Prefab引用**：在Inspector中为`_bulletPrefab`字段赋值
2. **设置射击频率**：通过`_fireRate`字段控制射击间隔（单位：秒）
3. **使用时间累加**：在Update()中检查`Time.time >= _nextFireTime`
4. **更新射击时间**：每次射击后设置`_nextFireTime = Time.time + _fireRate`
5. **实例化子弹**：在玩家位置实例化子弹Prefab

**最佳实践**：
- 使用`[SerializeField]`标记可配置的射击频率和子弹Prefab
- 在Shoot()方法中添加null检查，避免运行时错误
- 使用对象池（ObjectPool）管理子弹，优化性能
- 在子弹上添加Rigidbody2D和脚本控制移动方向
- 在Shoot()方法中添加日志，方便调试

**常见错误**：
- 只定义了Shoot()方法但没有在Update()中调用
- _bulletPrefab未在Inspector中赋值导致Instantiate失败
- 使用`InvokeRepeating()`但忘记在OnDisable()中CancelInvoke
- 射击频率设置过快导致性能问题
- 忘记设置子弹的移动方向和速度

**性能优化建议**：
- 使用对象池（ObjectPool）复用子弹对象
- 限制同时存在的子弹数量
- 使用协程替代Update()中的时间检查（对于复杂射击逻辑）
